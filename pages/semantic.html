Apa itu HTML, CSS, & JS

Apa itu HTML ?

Html sendiri merupakan singkatan dari hypertext markup language, merupakan sebuah (bukan termasuk bahasa pemrogramman) markup atau kerangka yang didalamnya bisa kita sisipkan css maupun javascript.

Apa itu CSS ?

Css (Cascading Style Sheets) adalah bahasa yang bisa kita sisipkan pada html. Css lah yang memberikan atau menggambarkan bagaimana elemen pada html akan ditampilkan. Pertama kali ditemukan pada 10 oktober 1994 oleh Håkon Wium Lie, sekarang css telah menjelma menjadi sebuah bahasa style sheet yang sangat fundamental penggunaannya.

Versi terakhir dari css adalah css3, selain itu css juga memiliki beberapa framework seperti bootstrap contohnya yang bisa kita gunakan untuk memudahkan kita dalam membuat sebuah website. Dengan menggunakan bootstrap, kode – kode css tadi sudah disiapkan kedalam sebuah kerangkan kerja yang bisa kita gunakan hanya dengan memanggil class maupun id. Penggunannya yang mudah dan juga sudah support responsive (tampilan dapat menyesuaikan pada perangkat yang lebih kecil) membuat bootstrap menjadi salah satu framework css primadona bagi front end developer.

Cara menyisipkan css sendiri ada 3, external internal dan inline. Dalam penggunaan external, kode kode css ditulis pada file terpisah dan pemanggilan pada html biasanya menggunakan class atau id. Sedangkan internal kode css biasa nya ditulis pasa dokumen html yang sama. Berbeda dengan keduanya, penulisan inline disematkan langsung pada objek atau elemen html.

Apa itu Javascript ?

Dikembangkan pada 1995 oleh netscape, javascript telah menjadi fenomena tersendiri dikalangan web developer belakangan ini. Bagaimana tidak? Penggunannya yang cukup mudah, banyaknya library/plugin yang bisa kita pakai gratis, framework yang terus bermunculan, ditambah lagi hadirnya node js (javascript yang bisa dioperasikan sebagai web server) membuat javascript semakin powerfull saja beberapa tahun ke belakang ini.

Salah satu kehebatan dari javascript adalah kemampuannya untuk membuat sebuah halaman website menjadi lebih interaktif bagi pengguna. Dengan meng klik tombol tertentu misalnya kita bisa menampilkan beberapa baris data tanpa harus memuat ulang keseluruhan halaman pada browser atau bahkan kita bisa memanfaatkan javascript untuk melakukan pertukaran data antara client dan server dengan memanfaatkan metode AJAX.

Javascript, sepertinya memang menjadi sebuah keharusan untuk dikuasai baik oleh front end developer maupun back end developer (mengingat adanya node js) untuk saat ini. Mengingat javascript yang menjadi salah satu bahasa pemrogramman terpopuler saat ini, saya rasa javascript akan bertahan dan terus digunakan dalam waktu yang lama.

Kesimpulan

Ketiga nya adalah bahasa yang berbeda namun saling berkaitan dan sama sama dijalankan pada sisi client, html css dan javascript merupakan bahasa yang wajib untuk dikuasai. Melihat ketiganya yang mudah untuk digunakan dan dipelajari serta semakin banyaknya komunitas dan pengguna, seperinya html css dan javascript akan dan tetap menjadi fundamental bagi front end developer.
